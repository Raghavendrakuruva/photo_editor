# photo_editor
Photo Editor Using OpenCV and Streamlit
📌 Project Description
This project is a web-based Photo Editor built using Python, OpenCV, and Streamlit.
It allows users to upload an image and apply various image processing operations such as resizing, brightness adjustment, filters, and effects in real-time.
The application provides an interactive UI where users can control all features using sliders and buttons.

🛠 Tools Used
Python
Streamlit
OpenCV
NumPy
Pillow
✨ Features of the Application
Core Features
Upload an image
Resize the image
Adjust brightness
Adjust contrast
Convert image to grayscale
Apply blur effect
Apply warm filter
Portrait-style background blur
Apply sharpen effect
Download edited image
Extra Features
Edge detection
Cartoon effect

cd photo-editor-opencv-streamlit
2. Install Dependencies
pip install -r requirements.txt
3. Run the Application
streamlit run app.py
4. Open in Browser
http://localhost:8501
